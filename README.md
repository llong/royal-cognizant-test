# Royal Caribbean & Cognizant Coding Assessment

This is a React Native application built as part of the coding assessment for Royal Caribbean and Cognizant.

## Features

- **Item List Screen**: Displays a list of names with the ability to add new items and remove existing ones. Tapping on an item navigates to the details screen.
- **Item Details Screen**: Shows the selected item's name and allows toggling favorite status, which persists across app sessions.

## Technologies Used

- React Native
- Expo
- TypeScript
- React Navigation
- AsyncStorage for persistence

## Getting Started

### Prerequisites

- Node.js (version 14 or later)
- npm or yarn
- Expo CLI
- Android Studio (for Android emulator) or Xcode (for iOS simulator)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd royal
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the Expo development server:
   ```
   npm start
   ```

4. Run on a device/emulator:
   - For Android: `npm run android`
   - For iOS: `npm run ios`
   - Or scan the QR code with the Expo Go app on your phone.

## Usage

- On the home screen, use the text input to add names to the list.
- Tap "Add" to add or remove the entered name.
- Tap "Delete" to remove the last item from the list.
- Tap on any name to view details and toggle favorite status.
